# Section 2

Basics: Variables, Data Types, Opearators and Functions

## Variables

A data container / data storage.

`let, const`

`var` - older version

```
let userName = "Iryna";

const totalUsers = 15;
```

### Variable names

1. camelCase - userName
2. JS is case sensitive
3. Only letters and digits \_ $
4. snake_case - user_name
5. Do not start with digits
6. No speacial chars
7. Keywords are not allowed

### Opearators
`+ - * ? %`
